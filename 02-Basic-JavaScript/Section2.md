# 자바스크립트 기초 - 변수, 자료형, 연산&함수

### 자바스크립트를 가져오는 방법

1. HTML파일 내부의 `<head>` 섹션에 `<script>` 태그를 추가

### 변수와 상수

1. 변수

- 일종의 데이터 컨테이너 저장소이자 값이 변할 수 있는 데이터 컨테이너
- 값이 변할 수 있음
- `let` 키워드 사용

2. 상수

- 값을 변경할 수 없음 -> 값이 변해서는 안됨
- `const` 키워드 사용

### 변수와 상수 네이밍 규칙

✅

- 끝에 숫자가 올 수 있음
- '$' 첫 글자 허용
- '\_' 첫 글자 허용

❌

- snake_case 사용 (비권장)
- 숫자로 시작할 수 없음
- '$', '\_' 외의 특수문자는 사용할 수 없음
- 변수 및 상수 선언 키워드를 변수명 또는 상수명으로 사용할 수 없음

1. camelCase

- 단어 내에 공란 허용되지 않음
- 소문자로 시작하며 이어지는 단어는 대문자로 작성
- 예: userName

---

### 함수

- 주변의 모든 컨텍스트에 접근이 가능함
- 단, 함수 내부에서 정의된 내용은 함수 안에서만 사용가능함
- return문 다음에 나오는 코드는 실행되지 않음 (return이 함수 실행을 종료시킴)

| 섀도잉 과정

- 다른 범위에 변수를 생성함 -> 생성된 변수는 전역 변수를 덮어쓰거나 제거하지 않고 공존함

---

### 주석 추가하는 방법

1. '//' 를 맨 앞에 쓰고 시작 -> 한 줄로 작성하는 주석에 적합함

- 예: `//This is a comment!`

2. '/\*\*/' 사용 -> 여러 줄로 작성하는 긴 주석에 적합함

- 예: `/* As a comment!
This is also a comment.
And this is! */`

---

### 불리언

- 참 혹은 거짓의 값
- 조건부 코드에서 주로 사용함

---

### 객체

- 키와 값을 쌍을 이루는 데이터 집합
- 중괄호(`{}`) 사용

### 배열

- 데이터의 목록
- 대괄호(`[]`) 사용
