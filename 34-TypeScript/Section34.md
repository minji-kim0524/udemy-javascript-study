# 타입스크립트

- 타입스크립트는 브라우저에서 실행할 수 없으며, 브라우저도 인식하지 못함
  -> 브라우저는 JavaScript만 인삭함
- 타입스트립트는 구형 브라우저에서도 실행 가능한 코드로 변환해줌

### 빌트인 기능

- 타입추론: 현재 작업 중인 타입을 추론함
- 타입변환: 타입스크립트의 추론을 오버라이드할 수 있는 작업

### 대표적인 타입의 종류

1. 문자열
2. 숫자
3. 불리언
4. 객체
5. 함수
6. undefined
7. void
8. any

### 타입별 참고사항

1. void vs. undefined

- undefined: return 값이 undefined 라고 받아들임 => return문은 있음
- void: 아무 값도 반환하지 않고자 할 때(실질적으로 undefined일 경우) return값이 존재하기는 한다는 뜻으로 받아들임 => return문 자체가 없음

2. any

- 신중히 사용해야하는 타입 << 최후의 타입임
- 모든 값에 대해 기본으로 설정되어 있으므로 타입을 이용해 작업하지 않음을 뜻함
  => 타입스크립트의 타입 지정을 통한 모든 이점을 포기한다는 의미와 같음

3. 객채

- 일반적으로 객체임을 알려주지만 관련 정보를 주지는 않음
