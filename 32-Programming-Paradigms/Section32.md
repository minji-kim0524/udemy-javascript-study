# 프로그래밍 패러다임 (Programming Paradigms)

| 코드를 작성하고 정돈하는 방식

## 프로그래밍 패러다임 종류

### 1. 객체 지향 프로그래밍

- 데이터와 논리를 객체나 클래스에 따라 정리
- 객체가 클래스를 기반으로 함 -> 객체의 프로퍼티에 데이터 저장가능
- 논리적인 개체에 따라 코딩을 진행함

### 2. 절자 지향 프로그래밍

- JavaScript 사용하기 시작했을 때 사용한 방법
- 간단하게 이어지는 단계와 작업의 실행을 사용 -> 코드의 실행 목록 생성
- 코드를 논리적인 그룹 || 데이터의 논리적인 조각으로 정리하지 않음 ❌
- 코드 구성시 객체 || 개체 사용하지 않음 ❌

### 3. 함수형 프로그래밍

- 가능하다면 코드를 정리하는데 명확히 구분된 순수 함수를 찾아냄
- 매개변수와 함수 사이에 데이터를 넘기고 함수는 코드를 정리하는 수단
