### Maps & Sets

자바스크립트의 주요 이터러블 데이터 구조 3가지

1. 배열(Arrays) 🌟

- 요소의 순서가 보장됨
- 중복이 허용됨

2. Sets

- 데이터 구조로 어떤 길이의 중첩 데이터도 저장함
- 요소의 순서가 보장되지 않음
- 중복이 허용되지 않음 ❌ ex: ID
- 인덱스 기반으로 접근하지 않음

3. Maps

- 데이터 구조로 어떤 길이의 키-값 데이터도 저장함
- 객체는 오직 문자열이나 숫자만을 키 또는 심볼형으로 가질 수 있는 반면, Maps는 아무거나 사용할 수 있음
- 객체에서는 순서가 보장되지 않는 반면, Maps에서는 순서가 보장됨
- 키는 중복이 허용되지 않지만, 값은 중복할 수 있음
- 키로 값에 접근함

### Maps vs. Objects

1. Maps

- 큰 규모의 데이터 사용시 활용
- 데이터를 자주 추가하거나 삭제할 때 유용함

2. Objects

- 작거나 중간 사이즈(30~40개의 키-값 쌍)의 데이터 사용시 활용
